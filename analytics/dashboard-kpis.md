# Dashboard KPI Priorities

## Top 5 (primary)

1. Users
2. Engaged sessions
3. Users by country and region
4. Users by device category
5. Top chapters by engaged sessions

## Extended book writer KPIs

- Chapter completion rate (`chapter_complete / chapter_view`)
- Code copy rate (`code_copy / chapter_view`)
- TOC interaction rate (`toc_interaction / chapter_view`)
- Trend over time (users + engaged sessions)
- Entry and exit chapter paths

## Required filters

- Date range
- `book_id`
- `content_group`
- Country
- Region
- Device category
